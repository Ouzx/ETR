# ETR
Evolution to Revolution

Evrimi kontrol altına alırsak neler olur?
ETR ile bu sorunun cevabını bulabilirsiniz.
Unity ile geliştirdiğim PC oyunu ETR henüz geliştirme aşamasında.

What happens if we take evolution under control?
You can find the answer to this question with ETR.
ETR, the PC game I developed with Unity, is still under development.
